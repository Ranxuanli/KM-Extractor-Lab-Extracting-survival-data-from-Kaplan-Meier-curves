Why this site exists

This project demonstrates how multimodal LLMs can help extract survival information from Kaplan–Meier plots when raw individual-level data are unavailable.

What is cached

Each run stores extraction JSON and a rendered plot so the workflow can be reproduced later without exposing any live API key.

Statistical caution

Pairwise and indirect comparisons shown here are approximate and intended for coursework demonstration, not definitive clinical inference.
